Quick Start Documentation

üìã Prerequisites

  • PowerShell 7+ installed
  • Microsoft Graph PowerShell SDK
  • Azure AD app registration with appropriate permissions
  • CSV file with employee data

üöÄ Installation

# Clone the repository
git clone https://github.com/vidal-renao/graph-employee-onboarding.git

# Navigate to project
cd graph-employee-onboarding

# Install dependencies
Install-Module Microsoft.Graph -Scope CurrentUser

‚öôÔ∏è Configuration

  1. Create Azure AD app registration
  2. Grant required Graph API permissions
  3. Update authentication parameters in scripts
  4. Prepare CSV file with employee data

‚ñ∂Ô∏è Usage

# Production execution
.\src\Onboard-Employees.ps1 -CsvPath "data/employees.csv"

# Offboarding
